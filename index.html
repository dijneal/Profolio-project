Featured Projects: What I've done so far

Pilates Site

This project was designed to showcase the exercise type known as Pilates. The site loads in the default view, but if the user clicks on the 'Change Theme' button, they can view it in dark mode. The photos are interactive and enlarge when being hovered over. There is also a description notating the origins of Pilates that is sourced from a linked WiKi article. Clicking the 'sign up' button enables the user to fill out a questionnaire which allows the locations of the nearest Pilates classes to be sent to their email.

Weather App

This project was designed to be a weather search engine. The user can enter a city into the search bar and get the current weather as well as the 6 day forecast. The humidity and wind will also be shown. This project uses live data and is linked to a weather API. The weather icons will also change depending on the temperature shown.

World Clock

This project was designed to show the different time zones of cities around the world. There are 3 default time zones shown, but the user can use the drop down menu to select from 3 more options. The user can also select 'Current Location' to see the time of the city that they are currently in. The app will not only show the time but the date as well. The homepage link can be used to return back to the default time zones.
